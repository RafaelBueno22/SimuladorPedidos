##🍕 Simulador de Pedido

Projeto utilizando **HTML, CSS e JavaSript**. O usuáro escolhe itens e o sistema calcula o total com desconto e entrega.

## ✅ Objetivo
- Trabalhar **condicionais (if)**
- Manipular dados
- Construir a lógica básica de algoritimos
- Conhecimentos utilizados: variáveis, operadores, condicionais e função.

##💻 Funcionalidades

- Seleção de itens
- Cálculo automático
- Desconto
- Taxa de Entrega
